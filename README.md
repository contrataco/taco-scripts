# Taco Scripts - NovelAI Utilities

A collection of NovelAI user scripts and applications. Each tool has been moved to its own dedicated repository for easier maintenance and installation.

## Projects

### Scripts

| Project | Description | Repository |
|---------|-------------|------------|
| **Auto Memory Manager** | Automatically manages NovelAI's Memory field by tracking events, characters, and compressing older content | [novelai-auto-memory-manager](https://github.com/contrataco/novelai-auto-memory-manager) |
| **Story Suggestions** | Generates story continuation suggestions after each AI response | [novelai-story-suggestions](https://github.com/contrataco/novelai-story-suggestions) |

### Applications

| Project | Description | Repository |
|---------|-------------|------------|
| **Scene Visualizer** | Electron app + companion script for generating AI images alongside stories | [novelai-scene-visualizer](https://github.com/contrataco/novelai-scene-visualizer) |

## Migration Notice

This repository previously contained all projects in a monorepo structure. As of January 2026, each project has been moved to its own repository for:

- Easier installation and updates
- Independent versioning
- Cleaner issue tracking
- Simpler contribution workflow

Please visit the individual repositories linked above for the latest code, documentation, and releases.
