# NovelAI Applications

Desktop and web applications that complement NovelAI.

## Available Apps

### scene-visualizer

Electron desktop application for displaying AI-generated scene images alongside NovelAI stories.

**Features:**
- Embeds NovelAI in a native desktop window
- Automatically generates images based on story content
- Supports multiple NAI Diffusion models (V3, V4, V4.5)
- Configurable generation parameters

**Requirements:**
- Node.js 18+
- NovelAI subscription with image generation access
- The `scene-visualizer-script` installed in NovelAI

**Quick Start:**
```bash
cd scene-visualizer
npm install
npm start
```
